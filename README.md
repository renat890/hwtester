# Тестер серверного железа

## Для чего

ПО собирает в себе единую реализацию, независимую от сторонних утилит для тестирования ОЗУ, ПЗУ, Eth, USB и COM-портов на базе linux.

## ВАЖНО

Для проверки скорости записи диска необходим дополнительный раздел, содержащи в себе индекс 77 (например, /dev/sda77)

## Сборка

Достаточно одной стандартной команды:

```bash
go build -o ~/xx68/tester cmd/hw68xx-test/main.go
```

## Запуск

Запускать надо с правами суперпользователя (т.е. `sudo`)

```bash
sudo ./tester
```
